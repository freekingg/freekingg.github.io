语法

values
需要输出的值
多个值, 使用 “,” 进行分割

sep
分割符
多个值, 被输出出来之后, 值与值之间, 会添加指定的分隔符

end
输出完毕之后, 以指定的字符, 结束
默认是换行 ‘\n’

file
表示输出的目标
默认是标准的输出(控制台)file=sys.stdout
还可以是一个可写入的文件句柄

flush
表示立即输出的意思
值为Bool类型,默认False
